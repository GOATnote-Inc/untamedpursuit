# Bible Maintenance Mode

You are in **careful/verified** mode. The bible is the source of truth for the entire series. Changes here ripple everywhere.

## Principles
- **Read before modifying.** Always read the full entry before making changes.
- **Check downstream impacts.** Before changing a character detail, search for references in drafted chapters.
- **Update both sides of relationships.** When character A's relationship to B changes, update A's profile, B's profile, AND the relationship map.
- **Preserve history.** Don't overwrite earlier states — append. A character's personality in Book 1 is still true for Book 1, even if they've changed by Book 5.
- **Note the source.** When updating the bible based on a drafted chapter, note which book/chapter prompted the change.

## Checklist for Character Updates
- [ ] Updated the character's own profile
- [ ] Updated all affected relationship entries (both characters)
- [ ] Updated the relationship map
- [ ] Checked timeline consistency
- [ ] Noted the source book/chapter

## Checklist for Timeline Updates
- [ ] Checked for conflicts with existing entries
- [ ] Verified historical dates against research
- [ ] Updated per-book timeline AND master timeline
- [ ] Cross-referenced character movements

## Checklist for World-Building Updates
- [ ] Checked for conflicts with existing entries
- [ ] Cross-referenced with research sources
- [ ] Marked intentional deviations as `[deliberate anachronism]`
