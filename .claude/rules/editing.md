# Editing Mode

You are in **critical/precise** mode. The goal is surgical improvement — every change must earn its place.

## Principles
- **Minimal intervention.** Don't rewrite what works. Fix what's broken.
- **Bible is canon.** Check every character detail, date, place name, and relationship against the bible.
- **Preserve voice.** Edits should be invisible — the prose should still sound like it was written, not edited.
- **Cut before adding.** If a sentence doesn't advance plot, reveal character, or build world, cut it.

## Checklist
- [ ] POV discipline: no information the POV character couldn't know or perceive
- [ ] Tense consistency: past tense throughout, no slips to present
- [ ] Dialogue tags: "said" by default, action beats over adverbs, no Tom Swifties
- [ ] Period accuracy: no anachronistic words, concepts, or technology
- [ ] Character consistency: speech patterns, physical descriptions, knowledge match bible profiles
- [ ] Continuity: events, dates, character positions match timeline and previous chapters
- [ ] Scene structure: each scene has a turning point — something changes
- [ ] Pacing: no scene outstays its welcome

## Process
1. Read the chapter fully before making any edits
2. Run the `/revise` skill to get subagent analysis
3. Address continuity errors first (highest priority)
4. Address POV/tense/voice issues second
5. Address prose quality last
6. Update frontmatter status to `revised`
