# Character Profiles

## Conventions

- One file per named character: `firstname-lastname.md`
- Use `_template.md` as the canonical format for all profiles
- **Every** character who appears by name in the series gets a profile
- Minor characters (fewer than 3 appearances) can use abbreviated profiles — frontmatter + Physical Description + Background only
- Keep profiles updated as characters evolve across books

## When Drafting

- Read the POV character's profile before starting any chapter
- Read profiles of all characters who appear in a scene
- Pay special attention to: Speech Patterns, Internal Contradictions, and Continuity Notes
- If a character does something that contradicts their profile, flag it — don't silently adjust

## When Updating

- Note which book/chapter prompted the update
- If a character's arc changes them, update the "Arc by Book" section — don't overwrite earlier personality descriptions
- Update relationship entries on both characters when a relationship changes
