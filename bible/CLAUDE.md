# Bible — Source of Truth

This directory is the **canonical reference** for the entire Untamed Pursuit series. Every fact about characters, places, events, relationships, and themes lives here.

## Rules

1. **Read before writing.** Always read the relevant bible entry before drafting a scene involving that character, place, or event.
2. **Bible wins.** If a draft contradicts the bible, the bible is correct. Flag the discrepancy — never silently change the bible to match a draft.
3. **Update both sides.** When modifying a character's relationship, update both characters' profiles AND `relationships/relationship-map.md`.
4. **Log changes.** When the bible is updated based on drafted content, note the source (book/chapter) in the entry.
5. **Earlier books take precedence.** In any factual conflict, the earlier book's established facts are canon.

## Structure

- `characters/` — One file per named character, using `_template.md` format
- `timeline/` — Master chronology and per-book timelines
- `world/` — Locations, politics, culture, technology, language
- `relationships/` — Character relationship map with dynamics and evolution
- `themes/` — Thematic threads tracked across the series
- `mysteries/` — Open plot threads, foreshadowing, and unresolved questions
