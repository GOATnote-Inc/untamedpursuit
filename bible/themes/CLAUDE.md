# Themes

## Purpose

Track the major thematic threads that run across the series. Themes give the story its deeper meaning and should be expressed through character action and imagery, never stated directly.

## Rules

- Themes are **shown, not told**. If you can point to a sentence that states the theme, rewrite that sentence.
- Each theme should manifest differently in each book — evolving, deepening, or being challenged.
- Track which characters and plot lines embody each theme.
- Note recurring images, symbols, and motifs associated with each theme.
