# Mysteries & Open Threads

## Purpose

Track every promise made to the reader: foreshadowing, unanswered questions, planted clues, and unresolved tensions. This is the series' narrative debt ledger — everything planted must eventually be paid off or deliberately subverted.

## Rules

- **Log immediately.** When a chapter introduces foreshadowing or raises a question, add it here before moving on.
- **Track status.** Every thread has a status: `planted`, `developing`, `resolved`, or `subverted`.
- **Note resolution.** When a thread pays off, record where and how.
- **Review before drafting.** Before starting a new book, review all open threads — some should advance, some should deepen, and a few might resolve.
- **Nothing disappears.** If a thread is abandoned, mark it `abandoned` with a note explaining why. Readers notice.
