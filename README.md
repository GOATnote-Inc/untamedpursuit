## About This Project

**Untamed Pursuit** is a 9-book historical fiction series spanning the
American frontier from 1846 to the 1880s. It follows Clara Chen, Eve Garrett,
and the network of extraordinary people they build across a continent.

This entire project — story, world-building, research, and process — is
being written in public. Every draft, every outline, every character profile
lives in this repository.
